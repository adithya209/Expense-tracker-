# 💸 Expense Tracker - Java Swing Project

This is a Java Swing-based Expense Tracker application that allows users to:

✅ Set a personal budget  
✅ Add, view, and delete expenses  
✅ Track remaining budget in real time  
✅ Receive feedback when budget is exceeded

## 👨‍💻 Technologies Used

- Java 8+
- Java Swing (GUI)
- Object-Oriented Programming
